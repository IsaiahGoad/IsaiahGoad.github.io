My Projects

Redefining AI & IT: BirdWatch

While BirdWatch can be expanded for multiple use cases, this project focuses on developing it as an on-device AI security and IT system for Dell AI laptops. The application leverages the Neural Processing Unit (NPU) to run pre-trained large language models (LLMs) locally for real-time network monitoring and analysis.

BirdWatch inspects network packets entirely on-device, using LLM reasoning to summarize traffic behavior and detect anomalies while maintaining privacy and responsiveness. BirdWatch frees the CPU and GPU, delivering efficient, intelligent, and private AI computing without relying on the cloud.
